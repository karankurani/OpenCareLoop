# Start Here

OpenCareLoop is a local workspace for organizing non-urgent health dossiers with
an AI coding agent.

## First Run

1. Unzip this folder.
2. Open the unzipped `OpenCareLoop` folder in Codex or Claude Code.
3. Ask the agent: `Set up OpenCareLoop and create a new person dossier.`
4. Keep new external records in `people/<person-slug>/raw-data-dump/` until the
   agent sorts and ingests them.

## Privacy

This workspace can contain sensitive medical information after you start using
it. Do not publish, commit, upload, or share a used workspace unless every
affected person has approved.
