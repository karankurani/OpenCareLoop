# OpenCareLoop

OpenCareLoop is a workspace for maintaining long-term health dossiers for one or more people with help from an AI agent.

It is designed for non-urgent use: organizing symptoms, medicines, records, timelines, summaries, and questions for future care conversations. It is not a medical device, clinician, emergency service, diagnosis engine, or treatment system.

## User Guide

The user-facing documentation lives in the Docusaurus site under [`docs-site/`](docs-site/).

When published with GitHub Pages, it will be available at:

https://karankurani.github.io/OpenCareLoop/

## Privacy

This repository may contain sensitive health information when used for real dossiers. Do not publish, commit, upload, share, or send personal medical records, prescriptions, lab reports, scans, IDs, or dossier files unless the affected person has approved that disclosure and you understand the privacy consequences.

The public docs site is intended to publish project guidance only. It should not include any `people/` dossier content or raw medical records.
