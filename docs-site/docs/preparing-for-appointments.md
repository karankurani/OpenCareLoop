# Preparing For Appointments

Before a visit, ask the agent to create a short appointment note from the current dossier.

The note should help the clinician see:

- The main reason for the visit.
- A brief timeline.
- Current medicines and recent changes.
- Relevant test results.
- What has already been tried.
- What helped or made things worse.
- The most important questions.

## Prompt to use

```text
Please prepare a concise appointment note from this dossier. Include the main concern, timeline, current medicines, relevant records, and the top questions for the doctor.
```

## After the appointment

After the visit, do a brain dump of the visit when information is still fresh.
