# Keeping Information Accurate

Accuracy matters more than polish. A short factual note with dates and uncertainty is better than a confident but vague summary.

## Use dates whenever possible

Use exact dates when known:

```text
Started metformin on 2026-05-03.
```

If the exact date is unknown, say so:

```text
Started sometime in early May 2026. Exact date unknown.
```
